# 💻 dlearning

Guía interactiva estilo Windows 7 Aero para aprender herramientas esenciales de desarrollo: Git, VSCode y Terminal. Navegación con sidebar colapsable, demos en vivo y código copiable.

![React](https://img.shields.io/badge/React-18-61dafb?logo=react)

## 🔗 Temas cubiertos

**Git:** Intro · Init · Staging · Branches · Merge · Remotes · Log · Undo · Git Flow · Tips

**VSCode:** Interfaz · Atajos · Extensiones · Terminal · Git · Debug · Snippets · Config · Multi-cursor · Tips

**Terminal:** Intro · Navegación · Archivos · Texto · Permisos · Pipes · Scripts · Variables · SSH · Tips

## 🛠️ Tecnologías

- **React 18** — Biblioteca de UI
- **Tailwind CSS v4** — Estilos utility-first
- **Vite** — Build tool
- **Lucide React** — Iconos

## 🚀 Ejecutar

```bash
npm install
npm run dev
```
